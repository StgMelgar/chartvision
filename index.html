Setup Guide

Up and running in minutes

ChartVision runs locally on your Mac or PC. You'll need a few free accounts to get started.

1

Get a Claude API Key

Sign up at console.anthropic.com and create an API key. ChartVision uses Claude Vision to analyze your charts. Add funds to your account (usage is typically $0.01–0.05 per scan).

2

Set Up TradingView

Open TradingView and create a 4-panel layout: 4H · 1H · 15M · 5M for your chosen symbol (QQQ, SPY, /NQ, etc.). Position the window exactly where ChartVision will capture it.

3

Download & Install ChartVision

Click the Download button below — we auto-detect your OS and serve the right installer. Native app, no Python or dependencies required.

Mac users:

1. Open the downloaded ChartVision-Mac.dmg file from your Downloads folder (double-click it).

2. A window will open showing the ChartVision app icon.

3. Drag the ChartVision icon onto your Applications folder (you can also drag it into the Applications shortcut in Finder's sidebar).

Windows users:

1. Open the downloaded ChartVision.exe from your Downloads folder.

2. If Windows SmartScreen appears, click More info, then Run anyway. (This happens because the app isn't yet signed by a paid developer certificate — the app is safe.)

4

(Optional) Connect TastyTrade

Enter your TastyTrade credentials in ChartVision Settings to enable live trade execution for options and futures. Paper trading mode is available without credentials.

5

Launch & Monitor

Mac — First launch (one-time setup):

Because ChartVision isn't yet signed with a paid Apple Developer certificate, macOS will block the first launch with a warning. Here's the 20-second fix:

1. Open Terminal. Quickest way: press Cmd + Space, type Terminal, press Enter.

2. Paste this one line into Terminal and press Enter:

xattr -cr /Applications/ChartVision.app && open /Applications/ChartVision.app

Nothing will print after you press Enter — that's normal. The command tells macOS to trust the app, then launches it. ChartVision should open within a few seconds.

After this first launch, double-clicking ChartVision from Applications will work normally — you never need to run this command again.

Windows — First launch:

Double-click ChartVision from your Desktop or Downloads. If SmartScreen appears, click More info, then Run anyway.

Once ChartVision is open:

1. Go to Settings and paste your Claude API key from Step 1.

2. Select your trading symbol (QQQ, SPY, /NQ, etc.).

3. Click the capture-region button and select your TradingView 4-panel layout.

4. Press Monitor — the AI will analyze your charts every 10 seconds.
